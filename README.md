# Shutters.social

Shutters.social is an upcoming image sharing network built on the AT Protocol.

## Requirements

- A working Rust installation (we recommend using [rustup](https://rustup.rs))
  - Cargo
  - Rust (Edition 2024)
- [Just](https://just.systems)
